**Comment utiliser le programme**

1- Double-cliquez sur le programme.

2- Choisissez la façon dont les données sont entrées dans le programme (automatique = 0; avec un fichier = 1) et appuyez sur Enter.

	a- Si les données sont générées automatiquement, choisissez les paramètres N(ombre de données), R(ang/Intervalle), D(ésordre, taux de) et appuyez sur Enter.

	b- Si les données sont chargées à partir d'un fichier, entrez le nom du fichier ou glisser le fichier dans la console et appuyez sur Enter.

3- Choisissez la façon dont les recherches sont entrées dans le programme (manuellement = 0; avec un fichier = 1) et appuyez sur Enter.

	a- Si les recherches sont chargées à partir d'un fichier, entrez le nom du fichier ou glisser le fichier dans la console et appuyez sur Enter.

4- Choisissez le nom du fichier de sortie avec son emplacement et appuyez sur Enter.

5- Choississez l'algorithme de recherche qui sera utilisé et appuyez sur Enter.

6- Le programme s'exécute. Si vous avez choisi d'entrer les données à la main, taper la valeur de chercher et appuyez sur Enter. Si la valeur existe,
   le programme retourne l'index de cette valeur, sinon elle retourne -1.

7- En mode manuel d'entrée de recherche, pour quitter le programme, entrer un nombre négatif et appuyez sur Enter.

8- Les statistiques s'affichent à l'écran à la fin de l'exécution. Ces statistiques sont aussi présentent dans le fichier de sortie 
   créé à la fin de l'exécution du programme.
